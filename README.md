# 光阴簿 PomoTasks

一个纯前端个人任务管理系统，集任务拆解、番茄钟、周视图日程、统计可视化于一体。打开即用，无需安装、注册或联网。

## 功能

### 任务管理
- 无限嵌套大任务 / 子任务，支持拖拽排序、合并拆分
- 完成百分比精确到 0.1%，进度条实时显示
- 任务分类：重要 / 日常 / 弹性，颜色标识
- DDL 截止日期，自动计算过期、今天、待定状态
- 预估工作量（分钟），按星级分类（🟢轻量 / 🟡中级 / 🔴困难）
- 实际花费时间自动追踪，番茄完成后累加
- 子任务权重按预估分钟自动分配

### 番茄钟
- 可自定义工作 / 休息时长
- 用真实时钟计时，切后台不丢精度
- 暂停 / 继续 / 提前完成 / 中止下次继续
- 完成后弹窗问进度，自动更新任务百分比

### 周视图日程
- 7 列日历，时间轴精确到小时
- 支持时间段任务和 DDL 标记（0 分钟 = 截止点）
- 同一时段重叠任务自动并排显示
- 当前时间线实时标注
- 可编辑名称、时间、时长、分类、备注

### 统计
- 番茄数、专注时长（今日 / 本周 / 本月 / 自定义）
- 活跃时段热力图（7 天 × 18 小时）
- 下钻明细：月 → 周 → 天 → 每小时
- 已完成任务荣誉墙

### 系统
- 数据自动保存到浏览器 localStorage
- 可选存为桌面 JSON 文件（浏览器关闭也不丢）
- 操作日志可导出 Markdown
- 支持备份 / 恢复 JSON

## 使用方式

双击 `index.html` 或者在浏览器打开即可。所有数据存于浏览器本地，不上传任何服务器。

## 技术栈

- 纯 HTML / CSS / 原生 JavaScript
- 零依赖，无框架，无构建工具
- File System Access API（可选桌面存储）
- Web Audio API（完成提示音）
- Notification API（定时任务提醒）

## 项目结构

```
pomotasks/
├── index.html          # 主页面：番茄钟 + 任务列表
├── calendar.html       # 日程：周视图日历
├── stats.html          # 统计：数据可视化
├── done.html           # 荣誉墙：已完成任务
├── style.css           # 共享样式
├── data.js             # 数据层：CRUD + 存储
└── app.js              # 界面层：渲染 + 计时 + 交互
```
